What is it?

The golden rectangle, also called a perfect rectangle, is a rectangle that can be divided into a square and another rectangle with with the same width:height ratio. This pattern can be reapeated infinitely with the smaller rectangle. This design can be seen everywhere from art to architecture to nature.

The Golden Rectangle

This site was built using golden triangle principles

The secret to the golden rectangle's magic is the ratio of it's height to it's width. This ratio is called the golden ratio.

Like many mathematical constants such as e and pi, the golden ratio is an irrational number, but it's value can be approximated as:

φ ≈ 1.61803

The Golden Ratio

This number can also be defined using things other than a rectangle. It can also be difined with the fibonacci sequence (1,1,2,3,5,8,13...). Specifically, the ratio of consecutive terms in the sequence approaches the golden ratio. (1/1, 2/1, 3/2, 5/3, 8/5,...144/89,...233/144). Alternatively the number can be defined as ((1+sqrt5)/2) or 1 + 1/(1+1/(1+1/(1+1/(...))))

Another cool part of the golden rectangle is that if you draw a 45 degree arc in each square then it creates a fibonacci spiral. This spiral is extremely similar to the spiral found in a nautilus shell
